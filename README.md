# Semana Omnistack 9.0 - AirCnC - Rocketseat

## Environment File
Rename .env_example to .env

## Database
Configure MongoDB URL in .env file
```
MONGO_URL=
```

## Backend
### Start the sever
```
cd backend
yarn dev
```


## Frontend
```
cd frontend
yarn start
```


## Mobile
```
cd mobile
yarn start
```
